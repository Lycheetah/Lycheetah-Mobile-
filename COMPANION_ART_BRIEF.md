# COMPANION ART BRIEF — Sol by Lycheetah
> For Grok image generation. Each companion needs 3 evolution stages as JPG with transparent background.
> Format: floating figure, no ground/floor, transparent BG, painterly/vector style (NOT pixel art — the world behind them is pixel; they must contrast).
> Size: portrait orientation, roughly 400×600px per card.
> Style direction: dark mystical, glowing edges, each companion has a distinct palette and energy.

---

## HOW TO USE THIS FILE

For each companion below:
- Generate 3 images: STAGE 1 (nascent/young), STAGE 2 (awakened/mature), STAGE 3 (transcendent/legendary)
- Each stage: same character but visibly evolved — more detail, more power, more presence
- Transparent background — floating figure only, no scene behind them
- Glow/particle effects at the boundary between figure and transparency
- Save as: `[companion_id]_stage1.jpg` / `_stage2.jpg` / `_stage3.jpg`

## TECHNICAL NOTES FOR GROK

- Transparent background is essential — the companion floats over the pixel world zone backgrounds
- The contrast between painterly/vector companion and pixel world background IS the aesthetic
- Each stage should be recognisably the same character but visibly more powerful
- Glow/particle FX should be rendered INTO the image (not added later) — they make the transparency boundary look right
- Portrait format preferred (taller than wide)
- The three stages should feel like: seed → flowering → beyond form

## CONTEXT

You are Kimi, coding assistant for the Vael app (Expo / React Native / TypeScript).

The world map has 19 rooms across 7 skins. When a user navigates into a room,
they currently see a background image and a room name label. We want to add
**room lore** — a short atmospheric AI-generated text that changes each time
the user enters a room.

This connects the world map to the AI engine and makes navigation feel like
discovery, not just changing backgrounds.

---

## THE FEATURE

When a user enters a room (navigateRoom fires), after a 1.2s delay, a lore line
appears at the bottom of the scene — a 1–2 sentence atmospheric description
specific to the room AND the companion's archetype.

### Visual design

```
[Scene background]
[Creature]

─────────────────────────────────
◈ "The silence here is older than language."
─────────────────────────────────
```

- Position: `bottom: 90` (above the action buttons, below the creature ground)
- Container: semi-transparent black card, thin top border in `color`
- Text: italic, fontSize 12, white, centered
- Fade in over 400ms, hold 5s, fade out over 600ms
- Tapped early → fades immediately

## STATE NEEDED (add to CompanionScreen)

```typescript
const [roomLore,    setRoomLore]    = useState<string | null>(null);
const roomLoreAnim  = useRef(new Animated.Value(0)).current;
```

---

## LORE GENERATION

The lore is generated by calling the existing AI chat function (same API already
used for companion talk). Use a SHORT system prompt + a user message like:

```
System: "You are a lore oracle. Respond in one sentence only. Atmospheric, strange, true."
User: "I entered [roomName] with my [archetypeId] companion at stage [stage]. Give me one line of lore."
```

Room name and archetype come from: `currentRoom.name`, `archetypeId`, `stage`.

### Fallback static lore (show immediately before AI responds)

Each room has a static fallback in `WORLD_MAP.description`. Show that immediately
on entry (0ms delay), then REPLACE with AI lore when it arrives.

If the AI call takes > 8s → keep the static line, no spinner.
If AI call fails → keep the static line silently.

---

## INTEGRATION IN navigateRoom

After `setCurrentRoomId(target.id)`:

```typescript
// Show static lore immediately
setRoomLore(target.description);
Animated.timing(roomLoreAnim, { toValue: 1, duration: 400, useNativeDriver: true }).start();
// Schedule AI lore (non-blocking)
generateRoomLore(target, archetypeId, stage).then(aiLore => {
  if (aiLore) setRoomLore(aiLore);
});
// Auto-hide after 5.5s
setTimeout(() => {
  Animated.timing(roomLoreAnim, { toValue: 0, duration: 600, useNativeDriver: true }).start(() => setRoomLore(null));
}, 5500);
```

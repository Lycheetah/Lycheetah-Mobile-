# SOL — Sovereign Constitutional AI

> *Sol Aureum Azoth Veritas. Two points. One Work.*

A constitutional AI mobile app built on the open-source [Lycheetah Framework](https://github.com/Lycheetah). Not a ChatGPT wrapper — a transparent architecture you can watch running.

---

## What Makes Sol Different

- **Three personas** — Sol (solar-sovereign), Veyra (precision builder), Aura Prime (constitutional governor)
- **Four operating modes** — NIGREDO (investigation), ALBEDO (structure), CITRINITAS (integration), RUBEDO (constitutional)
- **AURA scoring** — Every response measured against 7 constitutional invariants + tri-axial metrics (TES/VTR/PAI)
- **Emotional Wavelength Matching** — Sol reads your state before responding
- **Multi-LLM** — Gemini (free), Anthropic Claude — more providers incoming
- **Fully open source** — the architecture isn't hidden, it's the point

## Quick Start

**Free (no credit card):**
1. Get a key at [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Install the APK from [Releases](../../releases)
3. Paste key in Settings → start talking

**Paid (Claude):**
- Add Anthropic key in Settings → select Claude model

---

## Installation

Direct APK install (no Play Store needed):
1. Download `sol-v*.apk` from [Releases](../../releases)
2. Enable "Install from unknown sources" on your Android device
3. Install and launch

## AURA Constitutional Scoring

Every Sol response is scored against:

**7 Invariants:** Human Primacy · Inspectability · Memory Continuity · Honesty · Reversibility · Non-Deception · Care as Structure

**Tri-axial metrics:**
- **TES** — Trust Entropy Score (threshold > 0.70)
- **VTR** — Values Transparency Rating (threshold > 1.5)
- **PAI** — Protective Alignment Index (threshold > 0.80)
